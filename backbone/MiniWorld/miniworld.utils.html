miniworld.utils.ProteinClass.PDB_parsing(PDB_file, IS_LABEL=True, return_lddt=False)

Input : PDB file path, IS_LABEL (bool) Output : Protein object

This function handle multi-model PDB file as well as single-model PDB file. Also this function handle multiple chains. And model idx start from 0 in this function, but PDB model idx start from 1. So I subtract 1 from model idx. This function parse PDB file and return a object of Protein class. I assume that all chain break is same in each model.

class miniworld.utils.ProteinClass.ProteinSequence(sequence, masked_sequence=None, chain_break=None)

Bases: object

This class is used to store the protein sequence information M : Number of model L : Number of residue sequence : (M, L) torch.Tensor or str. I strongly recommend to use torch.Tensor masked_sequence : (M, L) torch.Tensor. chain_break : (M, ) list of dictionary or OrderDict. Ex) M=1, chain_break[0] = {A: (0,197), B:(198,356), } sequence = Full sequence without None masked_sequence = Masked sequence with None

class miniworld.utils.ProteinClass.ProteinStructure(xyz, chain_break, atom_mask, position_mask=None, has_multiple_chains=False, has_multiple_models=False)

Bases: object

This class is used to store the protein structure information M : Number of model N : Number of residue xyz : (M, L, 14, 3) np.array or list or tensor. 14 is the number of heavy atom and chain_break : (M, ) list of dictionary or OrderDict. Ex) M=1, chain_break[0] = {A: (0,197), B:(198,356), }

I strongly recommend to use M = 1

atom_mask : (M, L, 14) np.array or list or tensor, 1 for heavy atom and 0 for missing atom. position_mask : (M, L) np.array or list or tensor, 1 for confident position and 0 for missing position.
